NL4DV: Natural Language Toolkit for Data Visualization


NL4DV - Vega-Lite

A natural language interface for specifying and editing Vega-Lite charts. Show cases how NL4DV's visualization recommendations can be used to help novices get started with using Vega-Lite.

NL4DV - DataTone

A sample application illustrating how NL4DV can be used to recreate ambiguity widgets originally presented in the DataTone system developed by Gao et al.

NL4DV - MMPlot

A multimodal interface that allows interacting with scatter plots through touch and speech. This example illustrates how NL4DV can be used to augment existing interactions in a visualization system with voice.

NL4DV - Jupyter Notebook

For creating visualizations in Python without needing to learn about Python-based visualization packages.

Follow README for instructions.

Debugger

A debugging interface that breaks down NL4DV's output and highlights the underlying query interpretation logic.
